# つづもん Instagram ストーリー 予告画像 依頼 — Codex / image-2

**「開発中です」だけを伝える予告ストーリー3枚**。文字も含めて **1枚の絵として生成**する
（HTMLで文字を載せる方式はやめ、イラストの中に文字を焼き込む）。

## 大前提
- **サイズ 1080×1920（9:16）**。
- **上下それぞれ約250pxはInstagramのUI（アイコン・返信欄）に隠れる**。
  重要な文字・顔・主題はこの内側に収める。
- **中身は明かさない**。商品名（つづもん）・価格・機能・画面は出さない。
  「中学歴史の何か新しいものを作っているらしい」以上のことは伝えない。
- **見る人は中学生とその保護者**。堅い教材広告ではなく、**中学生がワクワクする冒険の入口**にする。
- **画像内の日本語はそのまま焼き込んでよい**（現在の生成技術で崩れない）。
  ただし生成後、**誤字・崩れを必ず目視で確認**すること。指定文言と1文字でも違えば作り直す。
- 3枚は**同じ画風・同じ色調で統一**（並べて見られるため）。実在の人物・商標は描かない。

## 共通の画風
水彩＋やわらかい線画の日本の教材イラスト風。クリーム（#fffdf8）と琥珀（#f59e0b）、
焦茶（#7c2d12）を基調にした温かいトーン。主人公は**中学生の男女どちらとも取れる、
特定の誰かに見えないキャラクター**（制服ではなくシンプルな私服＋リュック）。
出力先は `pdf-workbook/insta/assets/story-1.webp` … `story-3.webp`。

## 1枚目：共感（つらさ）
**焼き込む文字**（縦位置は中央より下・大きく）
```
テスト前に覚えて、
終わったら忘れる。
```
**プロンプト**
```
水彩とやわらかい線画で描く、縦長9:16の1枚絵。中学生くらいの子どもが机に突っ伏し、
開いた歴史の教科書とノートに埋もれている。頭の上には「大化の改新」「墾田永年私財法」
「参勤交代」などの歴史用語が、つかまえられない泡やシャボン玉のようにふわふわ浮かんで
すり抜けていく。用語の泡はうすい色で、いくつかは割れかけている。机の上には消しゴムの
かす、短くなった鉛筆、飲みかけのコップ。背景はクリーム色の壁で、窓の外は夕暮れ。
色調はクリーム色と琥珀色を基調にした温かいトーン。悲壮感ではなく「あるある」と笑える
やわらかさで描く。画面の中央より下に、手書き風の大きな日本語の文字で
「テスト前に覚えて、／終わったら忘れる。」と2行で入れる。文字は焦茶色で、
2行目の下に琥珀色の蛍光マーカーを引く。画面の上下それぞれ250pxには
主要な要素と文字を置かず、余白にする。1080×1920。
```

## 3枚目：予告（つづく）
**焼き込む文字**
```
中学歴史のための
あたらしいもの、つくってます。
つづく
```
**プロンプト**
```
水彩とやわらかい線画で描く、縦長9:16の1枚絵。画面の下から上へ、すごろくのような
丸いマスの道が奥へS字にうねって続いていく。手前のマスには「スタート」の木の立て札。
道の途中には縄文土器、埴輪、五重塔、天守閣、黒船、蒸気機関車の小さな挿絵が点在し、
いちばん奥は金色の光にかすんで見えない。マスの中身は空白のまま（数字や文字を入れない）。
道の手前に、リュックを背負った中学生の後ろ姿が小さく立ち、これから歩き出そうとしている。
空は夕暮れから夜へのグラデーションで、星が少し。全体に金色の光の粒。
色調は焦茶色と琥珀色を基調にし、1枚目2枚目より少し暗く落ち着いたトーンにする。
画面の中央やや上に、手書き風の日本語の文字で小さく「中学歴史のための」、
その下に大きく「あたらしいもの、つくってます。」と入れる。文字はクリーム色。
画面右下に小さく「つづく」と入れる。画面の上下それぞれ250pxには主要な要素と文字を
置かず、余白にする。1080×1920。
```

## 受け取り後の確認
1. 焼き込まれた日本語が**指定どおりか**（1文字でも違ったら作り直し）。
2. 実機のInstagramで**上下250pxが隠れても成立するか**（アイコン・返信欄の位置に文字が来ていないか）。
3. 3枚を並べて**画風と色調が揃っているか**。ズレていたら揃った2枚に寄せて残り1枚を再生成。
4. 商品名・価格・画面が**写り込んでいないか**（予告なので出さない）。

## 補足
文字だけのシンプルな面が必要になったときは `insta/make_story.py`（HTML→PNG）が使える。
今回のシリーズはイラスト主体なので、そちらは使わない。
