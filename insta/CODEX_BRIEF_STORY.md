# つづもん Instagram ストーリー 予告画像 依頼 — Codex / image-2

**「開発中です」だけを伝える予告ストーリー3枚**。文字も含めて **1枚の絵として生成**する
（HTMLで文字を載せる方式はやめ、イラストの中に文字を焼き込む）。

テーマは **「覚えても忘れる」ではなく「続かない」**。
問題集が三日で止まる、という中学生の“あるある”から入り、
最後に「"つづく"をつくっています」で予告して終わる。
（商品名は出さないが、後日「つづもん」と明かしたときに腑に落ちる伏線になる）

## 大前提
- **サイズ 1080×1920（9:16）**。
- **上下それぞれ約250pxはInstagramのUI（アイコン・返信欄）に隠れる**。
  重要な文字・顔・主題はこの内側に収める。
- **中身は明かさない**。商品名（つづもん）・価格・機能・アプリ画面は出さない。
- 見る人は**中学生とその保護者**。堅い教材広告ではなく、中学生が「自分のことだ」と笑える絵にする。
- **画像内の日本語はそのまま焼き込んでよい**（現在の生成技術で崩れない）。
  生成後、**指定文言と1文字でも違えば作り直す**。
- 実在の人物・商標・実在の教材は描かない。

## ★3枚の統一ルール（ここが今回の肝）
3枚は**同じ部屋で時間が進んでいく連続カット**として描く。並べたとき1本の物語に見えること。

| 要素 | 3枚共通の決めごと |
|---|---|
| 場所 | **同じ勉強部屋**。同じ木の机、同じ椅子、同じ窓、壁に同じカレンダー。カメラの位置もほぼ同じ |
| キャラ | **同じ中学生ひとり**。男女どちらとも取れる短めの髪、シンプルな私服（生成りのシャツ＋紺のズボン）、そばにリュック。制服にしない。特定の誰かに似せない |
| 画風 | 水彩＋やわらかい線画の日本の教材イラスト風。線は焦茶、塗りはにじみを残す |
| 色 | クリーム（#fffdf8）／琥珀（#f59e0b）／焦茶（#7c2d12）の3色を基調。**3枚とも明るいクリーム基調**にして、光の量だけを1→3で増やしていく |
| 光 | 1枚目＝夕方の弱い光 → 2枚目＝窓から差す明るい光 → 3枚目＝金色の光があふれる |
| 文字位置 | **3枚とも画面の下から1/3のあたりに左揃え**で置く。上下250pxには置かない |
| 文字の見た目 | **3枚とも同じ手書き風の書体・焦茶色**。強調したい1行にだけ**琥珀色の蛍光マーカー**を引く。書体・太さ・マーカーの太さを3枚で揃える |

出力先は `pdf-workbook/insta/assets/story-1.webp` … `story-3.webp`。

## 1枚目：あるある（続かない）
**焼き込む文字**
```
歴史のワーク、
三日でとまってる。
```
※2行目に琥珀色の蛍光マーカー。

**プロンプト**
```
水彩とやわらかい線画で描く、縦長9:16の1枚絵。中学生くらいの子ども（男女どちらとも
取れる短めの髪、生成りのシャツと紺のズボン、そばにリュック）が、勉強部屋の木の机に
ほおづえをついて、開いたままの問題集をぼんやり見ている。問題集は最初の数ページだけ
書き込みがあり、あとは白いまま。机には短くなった鉛筆、消しゴムのかす、閉じたままの
何冊かの問題集が積まれ、いちばん上の一冊にうっすらほこり。壁のカレンダーには
最初の三日だけチェックが付いていて、あとは空欄。窓の外は夕暮れで、光は弱い。
線は焦茶、塗りは水彩のにじみを残す。クリーム色と琥珀色と焦茶色を基調にした温かいトーン。
悲壮感ではなく「あるある」と笑えるやわらかさで描く。画面の下から3分の1のあたりに、
手書き風の大きな日本語の文字を左揃えで2行、「歴史のワーク、／三日でとまってる。」と
入れる。文字は焦茶色で、2行目に琥珀色の蛍光マーカーを引く。画面の上下それぞれ250pxには
主要な要素と文字を置かず、余白にする。1080×1920。
```

**焼き込む文字**
```

きみのせいじゃない。
```

**プロンプト**
```

目を輝かせている。窓から明るい光が差し込み、机の上の問題集のページが風にめくれて、
ページのあいだから光の粒と小さな紙吹雪が舞い上がる。積まれていた問題集のほこりは消え、
本が少し開いている。線は焦茶、塗りは水彩のにじみを残す。クリーム色と琥珀色と焦茶色を
```

## 3枚目：予告（"つづく"をつくってます）
**焼き込む文字**
```
その「つづき」を、
いま つくっています。
つづく
```
※2行目に琥珀色の蛍光マーカー。「つづく」は画面右下に小さく。

**プロンプト**
```
水彩とやわらかい線画で描く、縦長9:16の1枚絵。1枚目2枚目と同じ勉強部屋。同じ中学生が
椅子から立ち上がり、机の向こうを見ている。机の上の開いた問題集のページから、丸いマスが
連なるすごろくの道がふわりと立ち上がり、窓の外の空へ向かってS字にうねりながら
続いていく。道の途中には縄文土器、埴輪、五重塔、白い天守閣、黒船、蒸気機関車の小さな
挿絵が点在し、いちばん奥は金色の光にかすんで見えない。マスの中身は空白のままにして、
数字や文字を入れない。部屋には金色の光の粒があふれる。線は焦茶、塗りは水彩のにじみを
残す。クリーム色と琥珀色と焦茶色を基調にした温かいトーンで、3枚のなかで最も光が強い。
画面の下から3分の1のあたりに、手書き風の大きな日本語の文字を左揃えで2行、
「その「つづき」を、／いま つくっています。」と入れる。文字は焦茶色で、2行目に
琥珀色の蛍光マーカーを引く。書体・太さは1枚目2枚目とそろえる。画面の右下に小さく
「つづく」と手書き風で入れる。画面の上下それぞれ250pxには主要な要素と文字を置かず、
余白にする。1080×1920。
```

## 受け取り後の確認
1. 焼き込まれた日本語が**指定どおりか**（1文字でも違ったら作り直し）。
2. **3枚を並べて、同じ部屋・同じ子・同じ書体に見えるか**。ここが崩れると連作にならない。
   揃っている2枚に寄せて、残り1枚を再生成する。
3. 実機のInstagramで**上下250pxが隠れても成立するか**。
4. 商品名・価格・アプリ画面が**写り込んでいないか**（予告なので出さない）。

## 補足
文字だけのシンプルな面が必要になったときは `insta/make_story.py`（HTML→PNG）が使える。
今回のシリーズはイラスト主体なので、そちらは使わない。
