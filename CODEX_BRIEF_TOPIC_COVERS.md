# つづもん トピック単元表紙 量産ブリーフ（Codex用）

参考書の**細かい単元（トピック）**ごとの表紙画像を全92枚つくる。
（例: 歴史④の「聖徳太子と飛鳥文化」「大化の改新」…。参考書Web版の
「この単元でわかること」ページ相当を、表紙1ページに統合したもの）

デザインは**確定済みの章表紙テンプレートを完全踏襲**する:
- 正: `covers/html/unit-02.html` / `unit-11.html` ＋ `covers/css` の `cover.css`
- 見た目: A4縦・クリーム地に水玉＋薄い特大番号・角丸バッジ・マスキングテープで貼った
  少し傾いた写真カード・🎯この単元でわかることボックス・右横にマスコット＋吹き出し・
  最下部は波形オレンジ帯

## 作るもの

1. **生成スクリプト** `covers/gen_topic_covers.py`（Python）
   - `reference/{章}.json`（19冊）を読み、全トピックについて
     `covers/html/topics/{章番号}-{topicId}.html` を生成
   - Edge ヘッドレスで `covers/out/topics/{章番号}-{topicId}.png`（1240×1754）と
     同名 `.pdf`（A4）を出力
   - Edge コマンドは既存と同じ（README.md 参照。`--screenshot` / `--print-to-pdf`）
2. **全92トピックぶんの PNG + PDF**

## 各表紙の中身（データは reference/{章}.json から）

- バッジ: 「歴史 ④」（章の volume。既存バッジと同スタイル）
- タイトル: トピック名（例「大化の改新」）。長い場合は自動で少し縮小
- サブタイトル（unit-range の位置）: 「{章タイトル}／単元{トピック番号}」
  （例「古代の国家／単元2」）
- 背景の特大数字: トピック番号（例 2）
- 写真カード: `assets/reference/{t.image}`（**全92トピックに既存。新規画像生成はしない**）
- 🎯この単元でわかること: `t.learn` の項目（2〜3個・文が長いので折り返し可・
  フォントは章表紙より少し小さめでよい）。learn が無ければ sections の heading で代替
- マスコット＋吹き出し「この単元もがんばろう！」（章表紙と同じ）
- 波形帯: 「つづもん 中学歴史 まとめて復習ワーク」

## 実装の注意

- `cover.css` は共有し、トピック表紙用の差分だけ追加クラスで足す
  （既存の unit-02/11 の見た目を壊さない）
- 写真カードの傾きは章表紙と同様に交互（奇数トピック -2deg / 偶数 +2deg）で単調さを防ぐ
- learn の文はHTMLテキストで正確に。`**太字**` マークが入っていたら外してプレーンにする
- 画像は webp。Edge レンダリングなのでそのまま `<img>` で使える
- 章番号は reference ファイル名先頭2桁（例 `04-ancient-state.json` → `04`）
