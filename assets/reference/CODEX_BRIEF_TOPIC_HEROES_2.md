# 参考書トピック用ヒーローイラスト 残り39枚（Codex用ブリーフ・第2弾）

中学歴史の参考書で、各単元の冒頭に置く**円形アイコン風イラスト**の残り39枚を生成する。
出力先: `C:\Users\user\projects\education-apps\pdf-workbook\assets\reference\{ファイル名}.webp`
（PNGで生成してから同名の .webp に変換して保存してよい）

## ⚠️ 第1弾からの重要な変更（必ず読む）

第1弾の途中から **1200×800 の長方形**で作られた41枚があるが、これは**誤り**だった。
紙面のCSS（`generate_reference_book.py:264`）は

```css
.hero-fig img { width:36mm; height:36mm; object-fit:cover; border-radius:50%; }
```

で、**36mm四方の正円に切り抜いて表示する**。1200×800 で作ると中央の 800×800 だけが使われ、
**左右25%ずつが捨てられる**（例: hist01-timeline は左端の原始人と右端の現代人が消えて絵の意味が壊れている）。

**したがって今回は全枚 正方形 1024×1024 で作ること。**

## 最重要: 構図のルール（全39枚共通）

- **正方形 1024×1024**
- **背景は四隅まで塗りきる**。楕円のふちどり・白い余白・角の空白を一切作らない（円形トリミングは紙面側でやる）
- 主役は**中央の直径80%の円の中に完全に収める**（四隅は切れて消える前提）
- **主役は1つ**（または指定の2つ）。パッと見て「何の単元か」が0.5秒で分かること
- 人物は**胸から上（バストアップ）中心**、親しみやすいデフォルメ（写実にしない）
- 背景はうすい空色〜クリームのシンプルな風景・モチーフを**淡く**（主役より目立たせない）
- 文字・数字・透かし・署名は一切入れない

## スタイル（全枚共通）

- やわらかい線のフラットイラスト＋淡い水彩調。教材らしく明るく上品に
- 色数をしぼる。肌はナチュラル、輪郭はやわらかい焦げ茶
- **既存の `assets/reference/hist04-shotoku.webp` を必ず開いて見て、そのトーンに揃える**（これが基準）
- 39枚並んだときにトーンが揃って見えること
