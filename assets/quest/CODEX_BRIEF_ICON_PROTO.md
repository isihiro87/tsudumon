# 問題集UI アイコン イラストバッジ 試作 5枚（Codex / image-2）

つづもん問題集Webの「やり方をえらぼう」画面のカテゴリアイコン（現状はSVGライン）を、
codexのイラストで“おしゃれ”にできるか比較するための**試作**。統一トーンのアイコンセットを5枚。

## 保存先・手順
- 生成先: `C:\Users\user\projects\education-apps\pdf-workbook\tmp\iconproto\`（無ければ作成）
- 1枚ずつ生成→最新PNGを `tmp/iconproto/{出力名}.png` にコピー。全部できたら一覧出力して終了。

## スタイル（5枚で完全統一・最重要）
- **フラットなミニ・イラストアイコン**。角丸の正方形バッジの中に、そのカテゴリを表す**1つのシンプルな図**。
- 配色は**温かいブランド配色のみ**：ベース=クリーム(#fdf3e3)〜淡いアイボリー、主役=ブランド茶(#b45309)＋アクセントに琥珀(#f59e0b)。3色程度に絞る。
- やわらかい影・軽いグラデーションでほんの少し立体感（プレミアム感）。ただし派手にしない。
- **背景は角丸バッジの外を完全な透明**（PNGアルファ）。**文字・数字は入れない**。
- 5枚で線の太さ・余白・バッジの角丸・光の当たり方をそろえる（並べて完全に統一して見えること）。
- 正方形 512×512。図はバッジ中央に、周囲に均等な余白。
