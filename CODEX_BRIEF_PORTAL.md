# つづもん ポータル（歴史すごろく）画像依頼 — Codex / image-2

トップページ `output/web/index.html`（本番 `chatstudy.jp/tsudumon/`）を、中学生が「もう1マス進めたい」
と思える冒険マップ調にするためのイラスト。**生成モデル＝image-2**。

## 置き場所と反映
- 出力先：`pdf-workbook/assets/portal/`（ファイル名は下記のとおり）
- 反映：`python -X utf8 generate_tsudumon_portal.py` を実行すると `output/web/img/` へ自動コピーされる。
- **画像が無くても崩れない**（HTML 側で `onerror` により自動的に CSS だけの表示に戻る）。
  そのため未完成のまま置いておいても問題ない。届いた画像から順に豪華になる。

## 共通スタイル
- 水彩＋やわらかい輪郭線の日本の教材イラスト風。あたたかいクリーム／琥珀色（#b45309 / #f59e0b / #fffdf8）が基調。
- 中学生向け。かわいすぎず、幼稚にもならない。歴史の重厚さより「冒険のワクワク」を優先。
- **画像内に日本語を入れてよい**（現在の生成技術では崩れない）。ただし入れた文字は必ず目視で校正すること。
- 人物は特定の実在人物に似せない。

---

## ① portal-hero.webp （最優先）
- **サイズ 1600×600（8:3）**
- 内容：日本の歴史を横断する冒険マップの俯瞰。左から右へ時代が進む並び
  ―― 竪穴住居と土器 → 古墳 → 平安の寝殿造と桜 → 天守閣のある城下町 → 明治の蒸気機関車と赤レンガ →
  現代の高層ビルと新幹線。全体を1本の道（すごろくの道）がうねりながら貫く。空は晴天、雲は水彩のにじみ。
- **タイトル文字を焼き込む**：中央やや左に大きく「つづもん歴史 すごろく」、その下に小さく
  「中学歴史を楽しくマスター！」。木の看板か巻物の上に載せた立体的な題字にする。
- **下から25%は文字を置かない**（HTML 側でタイトル帯が重なるため、道や地面など情報の薄い部分にする）。
- **右端から15%には主役を置かない**（マスコットキャラが立つ位置）。

## ② portal-goal.webp
- **サイズ 640×480（4:3）・背景は白かクリームの単色**（切り抜き前提でもよい）
- 内容：開いた宝箱から光と金貨・巻物・王冠があふれ、紙吹雪が舞う。「ゴール！」の旗が一本立つ。
- 文字は「ゴール！」のみ、または文字なしでもよい。

## ③ portal-sign-{era}.webp （第2段階・後回しでよい）
時代エリアの見出しに使う木札。**サイズ 480×160**、背景透過。文字を焼き込む。
| ファイル名 | 焼き込む文字 | 添える小物 |
|---|---|---|
| portal-sign-ancient.webp | 原始・古代 | 土器・埴輪・稲穂 |
| portal-sign-medieval.webp | 中世 | 兜・弓・武士の旗 |
| portal-sign-earlymod.webp | 近世 | 天守閣・千両箱・扇 |
| portal-sign-modern.webp | 近代 | 蒸気機関車・洋館・ランプ |
| portal-sign-current.webp | 現代 | 東京タワー・新幹線・地球 |
