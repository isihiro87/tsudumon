# 中学歴史 資料画像 目録 — 3年間・全分野

**目的**: 我々の問題集（`pdf-workbook/`）の資料問題画像を、市販教材の水準に揃えるための参照台帳。市販の中学歴史教材から「**どの分野でどんな画像・資料が使われるか**」を全分野網羅で目録化した。

**分析ソース（全12分冊・約400ページを視覚精読）**:
- `新ワーク1〜6`（問題集＝画像が問題でどう使われるか）
- `必修テキスト1〜6`（解説書＝図版が最も豊富。各分野の定番資料の全体像）
- ※`ワーク1〜6`（旧版）は新ワークと内容重複が大きく、差分のみ後述の凡例で吸収

**画像の種類（凡例）**: 写真＝建築・仏像・遺物・人物 / 絵画＝屏風・絵巻・浮世絵 / 風刺画 / 史料＝古文書・碑文・宣言・条約 / 地図＝略地図・分布図・進路図・領域図 / 系図・組織図・関係図 / 模式図＝しくみ図・復元図 / グラフ / 表・年表

**★** = 複数の教材・ページで反復する最頻出の定番資料。**【現】** = 我々が現在すでに持っている画像。**【欠】** = 手本にあるが我々に無い（追加候補）。

**注意**: 本ファイルは目録・分析のみ。画像生成は行っていない。

## 全分野を貫く「画像の使い方」7原則（別ファイル `資料問題-画像の使われ方-分析.md` に詳細）

1. 画像は設問が名指しする具体物そのもの（飾りにしない）

5. 記述は資料を根拠に・語句を指定して書かせる
6. 一問一答には画像を置かない（役割分担）
7. 資料マッチング（「資料1〜3はA〜Dのどれと関係が深いか」）が全分野の花形
