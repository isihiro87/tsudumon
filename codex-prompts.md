# codex 画像生成プロンプト集（資料問題・図解用）

## 使い分けの方針

| 画像の種類 | 使うソース | 理由 |
|---|---|---|
| 実在の建築・仏像・絵画・写真資料 | Wikimedia Commons 等の実写（PD/CC0優先） | AI生成だと実物と違うものになり、テスト教材として致命的 |
| 地図・模式図・しくみ図・グラフ | **codex等のAI生成 or SVG自作** | オリジナルなので権利フリー。教材向けに単純化できる |

生成した画像は `assets/` に保存し、`books/{章}.json` の `"shiryo"` に登録すると
「E 資料問題」として紙面に載る（`credits.json` への追記は不要=自作扱い）。

## 共通スタイル指定（全プロンプトの末尾に付ける）

> 中学生向け教材の図解。白背景、印刷向けの高コントラスト、シンプルなフラットデザイン。
> 日本語ラベル付き。文字は大きくはっきりと。装飾や透かしなし。横長。

- 「高度経済成長期の実質GNP成長率の折れ線グラフ（1955〜1973、石油危機で終わる）」

## 登録方法

```json
"shiryo": {
  "topicId": [
    { "image": "ritsuryo-org.png", "caption": "律令国家の中央のしくみ",
      "questions": [ { "q": "図の①にあてはまる、政治を担当した役所は？", "a": "太政官" } ] }
  ]
}
```

登録後: `python -X utf8 generate_history_workbook.py` → Edge で PDF 化。
